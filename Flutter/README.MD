# Testes em Flutter

Este repositório centraliza os projetos desenvolvidos com **Flutter & Dart**, cobrindo desde a introdução ao framework até aplicativos com lógica de negócio, navegação entre telas e interfaces funcionais.

---

## Projetos

### 1. [Primeiro-Programa](https://github.com/UlissesCurvelloFerreira/Primeiro-Programa)

Projeto inicial de aprendizado em Flutter. Desenvolvido para compreender a estrutura fundamental do framework: funcionamento do `main.dart`, widgets básicos (`Text`, `Container`, `Column`, `Row`) e a organização padrão de um projeto Dart.

**Tecnologias:** Flutter · Dart

---

### 2. [App-Layout](https://github.com/UlissesCurvelloFerreira/App-Layout)

Aplicativo voltado para o estudo de composição de interfaces no Flutter. O foco foi compreender como estruturar e organizar telas utilizando os principais widgets de layout, como `Scaffold`, `AppBar`, `Column`, `Row`, `Padding`, `Expanded` e `Stack`.

**Tecnologias:** Flutter · Dart · HTML (web build)

---

### 3. [App-Jogo-da-Velha](https://github.com/UlissesCurvelloFerreira/App-Jogo-da-Velha)

Implementação do jogo da velha (Tic-Tac-Toe) para dois jogadores. O aplicativo gerencia o estado do tabuleiro, detecta automaticamente condições de vitória ou empate e permite reiniciar a partida.

**Funcionalidades:**
- Tabuleiro 3x3 interativo
- Alternância entre jogadores (X e O)
- Detecção de vitória e empate
- Reinício de partida

**Tecnologias:** Flutter · Dart

---

### 4. [Aplicativo_planetas](https://github.com/UlissesCurvelloFerreira/Aplicativo_planetas)

Aplicativo educativo com informações sobre os planetas do sistema solar. Cada planeta possui uma tela dedicada contendo dados relevantes e imagens ilustrativas. O projeto pratica navegação entre telas com `Navigator` e organização de dados estruturados.

**Funcionalidades:**
- Listagem dos planetas do sistema solar
- Tela de detalhes individual por planeta
- Exibição de imagens e informações descritivas

**Tecnologias:** Flutter · Dart

---

### 5. [App-Calculadora](https://github.com/UlissesCurvelloFerreira/App-Calculadora)

Calculadora com suporte às quatro operações matemáticas básicas. O projeto prioriza a construção de uma interface organizada com botões responsivos e gerenciamento de estado para exibição e atualização dos cálculos em tempo real.

**Funcionalidades:**
- Operações: adição, subtração, multiplicação e divisão
- Display de entrada e resultado
- Função de limpar (C)
- Interface responsiva

**Tecnologias:** Flutter · Dart
